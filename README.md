# MiniShell
Minimal representation of an actual shell terminal.

## Build
```bash
make
```

## Run
```bash
./minishell.exe
```

## Built-in commands
- `cd [dir]`
- `pwd`
- `quit`
- `jobs`
- `fg`
- `bg`
- `--help` / `--h`

## Notes
- Supports simple pipelines with `|`.
- Supports prompt update with `PS1=`.
